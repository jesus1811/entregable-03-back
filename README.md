# Documentacion

## Requisitos

Nodejs

Mysql

## Leventar Servidor

desarrollo

```
npm run dev
```

or

```
yarn dev
```

Produccion

```
npm start
```

## Base de datos

directorio

sql/database.sql, copiar y pegar en el localhost del mysql

## Mantenimiento

directorio
src/settings/database.js , modificar la cadena de conexion

```
host: "localhost",
user: "root",
password: "",
database: "bd_montalvo",
```
